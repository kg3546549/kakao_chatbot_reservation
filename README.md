# 모두예약

카카오톡 예약 메시지를 처리하고 예약 현황, 이력, 통계를 관리하는 Flutter 앱입니다.

현재 앱은 Android 예약봇 기능과 SQLite 로컬 우선 처리, Firebase 로그인,
멀티테넌트 동기화, 관리자 실시간 현황·이력·통계, FCM 푸시를 제공합니다.

Firebase 프로젝트 `releasenote-80bf5`에 Android 앱, 이메일/비밀번호 인증,
Firestore Rules·Indexes, Cloud Functions, App Check가 연결되어 있습니다.
